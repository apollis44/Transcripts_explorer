# TranscriptExplorer - Dash Application

This directory contains the interactive frontend for TranscriptExplorer. It is a [Plotly Dash](https://dash.plotly.com/) web application built with Bootstrap components. The app dynamically visualizes the data generated by the TranscriptExplorer pipeline.

## Features

- **Protein Search:** Search and select from a precomputed database containing all human proteins.
- **Membrane Topology:** Visualizes the predicted properties and membrane insertions along the amino acid alignments of different transcript isoforms.
- **Expression Analysis:** View dynamic expression distributions of a target protein's transcripts. You can select specific cancer types or healthy tissues to narrow down the comparative data (powered by TCGA and GTEx datasets).

## Prerequisites

Before running the application, ensure you have generated the required data files by running the `generating_necessary_files.py` pipeline in the root repository directory. The pipeline will populate the `files_for_plots` directory in this app folder with the datasets.

## Running the App

From within the `app` directory, run:

```bash
python app.py
```
